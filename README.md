# React_Project
#### TMDB API를 활용한 React 기반의 SPA 프로젝트로, 게시글 등록 및 관리 기능을 포함한 CRUD 구현

## 기능
+ #### TMDB API를 연동하여 Route 기반의 단일 페이지 애플리케이션(SPA) 구성
+ #### 브라우저의 LocalStorage를 이용한 데이터 저장 기능 적용
+ #### React의 Hook 함수 사용을 통한 게시글 및 검색창 상태 관리
